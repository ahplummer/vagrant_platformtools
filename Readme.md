# What is this?

This is a recipe that stands up a VirtualBox VM based on a Pop!_OS distribution, along with some interesting developer tools.

## Requirements

* VirtualBox
* Vagrant
* Ansible

## Get it working

* Stand up the VM in the cloned directory:
```
vagrant up
```
* Turn it off 
```
vagrant halt
```
* Re-provision
```
vagrant provision
```
* Destroy things with fire
```
vagrant destroy
```
